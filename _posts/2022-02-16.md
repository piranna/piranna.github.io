Git can be configured to use SSH in place of HTTPS for URLs matching a given
prefix. To use SSH for all GitHub access, add these lines to `~/.gitconfig`:

```gitconfig
[url "ssh://git@github.com/"]
  insteadOf = https://github.com/
```
